# SkillForge: Create, Lint, and Ship Viral AI Skills

An open-source starter kit + linter + gallery for the new wave of LLM “skills,” helping creators turn prompts, workflows, and templates into polished, shareable GitHub repos in minutes.

## Why This Repo

This project is generated by a trend-driven pipeline. It targets **Indie hackers, prompt engineers, educators, content creators, and developers building reusable Claude/Codex/GPT skills** and focuses on the niche: **AI agent skills tooling and distribution**.

## Key Features

- One-command CLI to scaffold a Claude/Codex/OpenAI-compatible “AI skill” from templates for content, research, slides, study notes, and code review
- Skill linter that checks README clarity, prompt injection risks, required inputs/outputs, citations policy, anti-fabrication rules, and portability across LLMs
- Local preview runner: test a skill against sample tasks and generate a shareable HTML report with before/after outputs
- Auto-generate GitHub-ready assets: badges, demo GIF script, topic tags, examples, and a polished README
- Community gallery generated from a simple skills.json file so contributors can add their own skills via PR
